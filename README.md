# ResumeIQ – AI-Powered Resume Screening & Skill Matching System

ResumeIQ is a role-based web application that intelligently analyzes resumes for **ATS compatibility** and **skill–role alignment** using **NLP and semantic similarity techniques**.  
It supports both **candidates** and **recruiters**, enabling smarter resume evaluation and role-based candidate classification.

---

##  Key Features

###  Candidate Side
- Resume upload and parsing
- ATS compatibility scoring
- Skill extraction and gap analysis
- Role fit percentage
- Suggested alternative roles

###  Recruiter Side
- Upload multiple resumes
- Job description–based matching
- Semantic skill matching using embeddings
- Candidate ranking by role suitability
- Best-fit role classification

---

##  How It Works

Resume Upload
↓
NLP Preprocessing & Skill Extraction
↓
ATS Analysis (Rule-Based)
↓
Semantic Matching (Sentence Transformers)
↓
Role Fit Scoring & Classification
↓
Dynamic Results (Candidate / Recruiter View)


---

##  Tech Stack

- **Backend:** Python, Flask  
- **NLP:** Rule-based NLP, text preprocessing  
- **AI:** Sentence Transformers, cosine similarity  
- **Frontend:** HTML, CSS, JavaScript (dynamic rendering)
