# ResumeIQ – AI-Powered Resume Screening & Skill Matching System

##  Project Description

**ResumeIQ** is a role-based web application that intelligently analyzes resumes for **ATS compatibility** and **skill–role alignment** using **NLP and semantic similarity techniques**.

It supports both **candidates** and **recruiters**, enabling smarter resume evaluation, role-based candidate classification, and efficient shortlisting through AI-powered insights.

---

##  Key Features

###  Candidate Side
- Resume upload and parsing
- ATS compatibility scoring
- Skill extraction and gap analysis
- Role fit percentage for a selected target role
- AI-detected best-fit role
- Resume improvement recommendations
- Visual insights for ATS score and role fit

###  Recruiter Side
- Upload multiple resumes
- Job description–based matching
- Semantic skill matching using embeddings
- Candidate ranking by role suitability
- Best-fit role classification per resume
- Resume viewing directly from results

---

##  How It Works
Resume Upload
↓
NLP Preprocessing & Skill Extraction
↓
ATS Analysis (Rule-Based)
↓
Semantic Matching (Sentence Transformers)
↓
Role Fit Scoring & Classification
↓
Dynamic Results (Candidate / Recruiter View)

##  Tech Stack

### Backend

- Flask

### NLP & AI
- Rule-based NLP
- Text preprocessing
- Sentence Transformers
- Cosine similarity

### Frontend
- HTML
- CSS (Tailwind CSS)
- JavaScript
